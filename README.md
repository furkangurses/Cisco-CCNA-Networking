# CCNA Networking Labs

This repository contains a collection of networking labs created while studying for the Cisco CCNA certification.

The goal of these labs is to practice real-world networking configurations and strengthen fundamental networking concepts using Cisco technologies.

## Topics Covered

- VLAN Configuration
- Inter-VLAN Routing
- Static Routing
- OSPF Routing
- DHCP Configuration
- NAT (Network Address Translation)
- Access Control Lists (ACL)
- Basic Network Troubleshooting
- Switch and Router Configuration

## Lab Structure

Each folder in this repository represents a separate lab and includes:

- Network topology
- Device configurations
- Command explanations
- Screenshots (if applicable)

## Purpose

These labs were built to gain hands-on experience with networking concepts commonly required for the **CCNA certification** and real-world network administration.

## Tools Used

- Cisco Packet Tracer
- Cisco CLI
- Basic networking troubleshooting commands
